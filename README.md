## 环境准备

安装 `node_modules`:

```bash
npm install
```

or

```bash
yarn
```

## 提供的脚本

### 启动项目

```bash
npm start
```

### 打包项目

```bash
npm run build
```

### 检查代码风格

```bash
npm run lint
```

You can also use script to auto fix some lint error:

```bash
npm run lint:fix
```

### 测试代码

```bash
npm test
```
